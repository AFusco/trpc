## What is RPC? What mindset should I adopt?

- don't think about HTTP/REST implementation details
- it's just a function call, though query/mutation are useful added concepts
- Link to RPC Spec

## Vocabulary

Below are some terms that are used frequently in the tRPC ecosystem. We'll be using these throughout the documentation, so it's good to get familiar with them.
